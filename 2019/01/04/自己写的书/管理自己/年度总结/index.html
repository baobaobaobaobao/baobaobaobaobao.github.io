年度总结

发表于 2019-01-04 | 分类于 管理自己 |

姗姗来迟的年度总结，今年是个“长见识”的一年。做了很多杂事，去了挺多地方，
认识了不少人。红红火火恍恍惚惚，又过了一年春夏秋冬。

1 今年

1.1 首推哲学

无论做什么。哲学绕不过去，正如凡事问一问有没有意义一样。万物的知识体质从
哲学中诞生，在哲学中寻找最后的归宿。不学点哲学，无法面对我背过的马克思主义。
今年侥幸读过的杂书有十多本。

1.2 进入新的环境

到了更热闹的城市，认识了很多优秀的人，无论在知识、健身、人品方面都远胜
于我，一直深感自己的菜鸡。

1.3 参加过一份工作

参加过一份本专业正式的工作，教会了我很多。也开始明白基层科研的欢乐。

1.4 认真培养一份“爱好”

“爱好”这东西，可能就是工作之余可以玩一下午都不厌烦的事情。开始认真培养
了2个爱好，希望能发芽。

2 展望明年

2.1 专业技能

专精一门，无间断学习。发论文。

2 .2英语

希望能跟随@嘉晨的脚步，坚持打卡半年口语学习，以达到和外国人正常交流水平。

2.3 爱好

期望羽毛球能达到业余运动员水平，能打到市级比赛。期望可以把所经历的终极问题
写成一个故事并发表。期望读完30本书。

2.4 坚持理财实践

开始从书籍理论学习到实践，希望未来更好。

2.5 深度思考

无论什么时候，都不要忘了深度思考所有事物发生的背后逻辑。私以为这是一个普通人变
聪明一点的唯一方法。

2.6 感恩他人

确确实实发现，人生的路上。总会有人给你指路，一旦你有了疑问，先思考再发问。
这几年来，我问过很多问题，或幼稚或深刻。但无一例外，都有人或者书指路，在此
特别都要感谢每个人我认识的人。你们对我的人生观、价值观都产生巨大影响，虽无以回报，
但时常心心念念。

3 总结

今年也是一个“波澜壮阔”的一年，在“学习就是使命”的指导思想下，立足于20余年的
砥砺前行，坚持主观能动性和客观规律性，离实现精神和物质的极大丰富更进一步。但梦想
和现实仍有巨大鸿沟，唯物与唯心之间有时难以决策，更需要以深度思考、坚韧不拔的毅力
来夺取2019之胜利！
